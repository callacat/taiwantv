# TaiwanTV — 台湾电视直播代理服务

给 TVBox/影视仓 提供台湾电视频道的免代理接口。

## 问题

`t.freetv.fun` 的 M3U 源需要走代理才能访问，且流经过 Cloudflare + 本地代理双重中转非常卡。

## 方案

```
你的海外 VPS (Docker)                     老人手机
┌──────────────────────────┐              ┌────────────────┐
│  Node.js 服务              │              │ 影视仓 App      │
│  ├─ 拉取 freetv.fun 的源  │  ──JSON──→  │                │
│  ├─ HLS 代理转发流        │              │  填订阅地址:     │
│  └─ 输出 TVBox 格式 JSON  │              │  http://你VPS   │
│                           │              │  IP:3000/       │
│  freetv.fun               │              │  tvbox.json     │
│    ↑ VPS海外直连，低延迟   │              │                │
└──────────────────────────┘              └────────────────┘
```

## 部署

```bash
docker compose up -d
# 订阅地址: http://你的VPS_IP:3000/tvbox.json
```

## 影视仓配置

1. 手机安装影视仓
2. 设置 → 配置地址 → 输入 `http://你的VPS_IP:3000/tvbox.json`
3. 回到首页 → 点"电视" → 选"台湾"分组 → 点频道播放

## 源维护

后端每 3 小时自动拉取最新源，无需手动操作。
